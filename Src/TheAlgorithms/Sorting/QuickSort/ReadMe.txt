- Write a program which will sort an unsorted array using quick sort technique.

Input  = {2, 5, 7, 9, 8, 3}
Output = {2, 3, 5, 7, 8m 9}


QuickSort: It is devide and conqure algorithm. It pick an element as an pivot and partition arond that pivot.

Pivot Element may be
	i) first element
	ii) Last element
	iii) a random element
	iv) midiun element


Time Complexity:  O(n^2)
