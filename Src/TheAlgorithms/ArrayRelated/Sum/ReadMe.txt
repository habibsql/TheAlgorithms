- How to find all pairs on integer array whose sum is equal to given number?

- Details: 
You have given an integer array and a number, you need to write a program to find all elements in
the array whose sum is equal to the given number. Remember, the array may contain both
positive and negative numbers, so your solution should consider that

- Input1: Sum = 20
- Input2: {20, -10, 30, -10, 10, -20, 50, 30}
- Output: {-10, 30} {30, -10} {-10, 30}
